## Usage

### Camera View Control

rotation: `Drag`

position: `Shift` + `Drag`

## Local Execute

### Setup

Install NodeJS

```sh
winget install -e --id OpenJS.NodeJS
```

Clone

```sh
git clone https://github.com/CaseyNelson314/EFSim.git
cd EFSim
npm install --save-dev
```

### Launch local server

```
npm run dev
```
